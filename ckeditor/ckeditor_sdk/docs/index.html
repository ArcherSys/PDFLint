CKEditor 4 Documentation

CKEditor is a browser-based WYSIWYG text editor, which means the text being edited looks as similar as possible to the results users have when publishing it. It brings to the web common word processor features found in desktop editing applications like Microsoft Word and OpenOffice.

Contribute

CKEditor is an Open Source project so your contribution is welcome. Fell free to report bugs or improve the code on Github.
Since CKEditor is localized, you can also help translating it. You don't have to be a programmer to contribute!
